# EcoToxLGB v2.0 详细使用说明

## 1. 环境配置

```bash
# 推荐使用conda
conda create -n ecotox python=3.11
conda activate ecotox
conda install -c conda-forge rdkit lightgbm
pip install numpy scikit-learn pandas matplotlib
```

## 2. 特征计算（247维）

特征构成：
- **42 SMARTS子结构**：苯环、羧酸、磺酸、卤素、酰胺等化学片段
- **166 MACCS指纹**：MDL公共化学结构指纹
- **29理化描述符**：MW、LogP、TPSA、Chi指数、Kappa指数等
- **12系统发育编码**：物种类别 one-hot + 跨物种毒性信息
- **4终点编码**：LC50/EC50/NOEC/LOEC

## 3. 模型加载与预测

```python
import pickle, lightgbm as lgb
from code.qsar_prepare import compute_ecotox_features

# 加载模型
model = pickle.load(open('trained_model/ecotoxlgb_v2_final.pkl', 'rb'))

# 计算特征并预测
# 注意：需要指定物种和终点
smiles = "CC(C(F)(F)F)(C(F)(F)F)O"
features = compute_ecotox_features(smiles, species="Fathead Minnow", endpoint="LC50")
pred = model.predict(features.reshape(1, -1))[0]
print(f"EcoToxLGB预测: {pred:.3f} log(1/LC50)")
```

## 6. 注意事项

- 模型特征包含物种和终点编码，预测时需提供这些信息
- 物种编码缺失时使用默认值，但精度会下降
- 该模型为通用模型（覆盖8物种4终点），对特定物种的精度可能低于专属模型
