# PSF-master 详细使用说明

## 1. 环境配置

```bash
# 创建虚拟环境（推荐）
python -m venv psf_env
source psf_env/bin/activate  # Linux/Mac
# 或 psf_env\Scripts\activate  # Windows

# 安装RDKit（核心依赖）
# 推荐使用conda安装RDKit
conda install -c conda-forge rdkit
# 或 pip install rdkit-pypi

# 安装其他依赖
pip install numpy scikit-learn matplotlib pandas
```

## 2. 特征计算

```python
from code.psf_master import psf_master
from rdkit import Chem

# 单个化合物
smiles = "CC(C(F)(F)F)(C(F)(F)F)O"
mol = Chem.MolFromSmiles(smiles)
features = psf_master(mol)
print(f"15维PSF特征: {features}")
# [LogP, MW, HBA, HBD, TPSA, RotB, HeavyAtoms, F_count, F_ratio, ...]
```

## 3. 模型加载与预测

```python
import pickle, numpy as np
from code.psf_master import psf_master
from rdkit import Chem

# 加载模型和标准化器
with open('trained_model/psf_gbr_model.pkl', 'rb') as f:
    model = pickle.load(f)
with open('trained_model/scaler.pkl', 'rb') as f:
    scaler = pickle.load(f)

# 预测
smiles = "CC(C(F)(F)F)(C(F)(F)F)O"
mol = Chem.MolFromSmiles(smiles)
features = psf_master(mol).reshape(1, -1)
features_scaled = scaler.transform(features)
log_lc50 = model.predict(features_scaled)[0]
print(f"预测 log(1/LC50) = {log_lc50:.3f}")
```

## 4. 批量预测

```python
python predict.py --input data/new_pfas.csv --output results.csv
```

输入的CSV应包含 `smiles` 列。

## 5. PSF-master 15维特征说明

| 索引 | 特征 | RDKit计算方式 |
|:---:|:----|:-------------|
| 0 | LogP | Descriptors.MolLogP(mol) |
| 1 | MolWt | Descriptors.MolWt(mol) |
| 2 | NumHAcceptors | Descriptors.NumHAcceptors(mol) |
| 3 | NumHDonors | Descriptors.NumHDonors(mol) |
| 4 | TPSA | Descriptors.TPSA(mol) |
| 5 | RotatableBond | Descriptors.NumRotatableBonds(mol) |
| 6 | HeavyAtomCount | mol.GetNumHeavyAtoms() |
| 7 | F_count | 遍历原子，AtomicNum==9计数 |
| 8 | F_ratio | F_count / HeavyAtomCount |
| 9 | O_count | 遍历原子，AtomicNum==8计数 |
| 10 | RingCount | rdMolDescriptors.CalcNumRings(mol) |
| 11 | CF3_count | MolFromSmarts('C(F)(F)F')匹配数 |
| 12 | halogen_count | F+Cl+Br+I原子总数 |
| 13 | FractionCSP3 | rdMolDescriptors.CalcFractionCSP3(mol) |
| 14 | MolMR | Descriptors.MolMR(mol) |

## 7. 注意事项

- 模型仅对PFAS类化合物有效，一般有机物的预测不可信
- 预测值附带AD检测标记 `in_ad`，仅当 `in_ad=True` 时预测可信
- 如需95%置信区间，使用 `predict.py --conformal` 参数
- 更详细的AD和Conformal分析见论文 §2.4.4
